# Sales Performance & Forecasting Dashboard

## Overview
This project analyzes historical sales data and presents key business insights using an interactive Power BI dashboard. It includes time-series analysis and short-term sales forecasting to support decision-making.

## Tools Used
- Power BI
- Excel

## Key Features
- KPI tracking for sales, profit, quantity, and delivery performance
- Sales analysis by region, category, and customer segment
- Time-series sales trend analysis
- 15-day sales forecasting with confidence intervals
